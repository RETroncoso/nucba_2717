Como usar CSS

Para usar CSS, debemos importarlo en nuestro HTML.

<link rel="stylesheet" href="style.css">

En el head usar la etiqueta de style

<style> p { color: red;} </style>

En la misma etiqueta

<p style="color:red;">Texto </p>
